# Engineering College Admission Predictor (CAP)

A **DBMS-based application** that predicts eligible engineering colleges during the **CAP admission process** using **previous year cutoff trends**.

The system focuses on selected colleges in **Pune, Mumbai, and Sangli**, and accepts **JEE / MHT-CET scores** along with **gender and category** inputs to provide accurate, data-driven predictions.

---

## Tech Stack
- **Java** – Application logic  
- **JDBC** – Database connectivity  
- **SQL (MySQL)** – Cutoff data storage & filtering  

---

## Highlights
- CAP-based college prediction  
- Region-specific results  
- Category & gender-based cutoff analysis  
- Well-normalized database design  
- Query-driven, accurate outputs  

---

## Use Case
Helps students make **informed admission decisions** using reliable DBMS-backed predictions.
